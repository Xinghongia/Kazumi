# 第11章 UI 组件与页面

> Kazumi 的 UI 层采用 Widget 组件化设计，基础组件在 `bean/` 目录下，
> 页面组件在 `pages/` 目录下。组件复用率很高，NetworkImgLayer 几乎无处不在。

## 11.3 NetworkImgLayer — 最复用的组件

> 源码：`lib/bean/card/network_img_layer.dart`

整个项目复用率最高的组件，几乎所有卡片都用它加载网络图片。

```dart
class NetworkImgLayer extends StatelessWidget {
    final String? src;        // 图片 URL
    final double width;       // 宽度
    final double height;      // 高度
    final String? type;       // 类型：'avatar'(圆形), 'emote'(无圆角), 默认(带圆角)

    Widget build(BuildContext context) {
        return ClipRRect(
            borderRadius: BorderRadius.circular(
                type == 'avatar' ? 50 : type == 'emote' ? 0 : StyleString.imgRadius.x,
            ),
            child: CachedNetworkImage(
                imageUrl: imageUrl,
                memCacheWidth: memCacheWidth,   // 内存缓存尺寸优化
                memCacheHeight: memCacheHeight,
                fit: BoxFit.cover,
                placeholder: (context, url) => placeholder(context),  // 加载占位
                errorWidget: (context, url, error) => placeholder(context),  // 错误占位
            ),
        );
    }
}
```

### 内存缓存优化

```dart
// 根据宽高比决定缓存维度，减少内存占用
if (aspectRatio > 1) {
    memCacheHeight = height.cacheSize(context);  // 横图只缓存高度
} else if (aspectRatio < 1) {
    memCacheWidth = width.cacheSize(context);    // 竖图只缓存宽度
}
```

### Hero 动画支持

```dart
// 静态方法，供 Hero 的 flightShuttleBuilder 使用
static Widget heroFlightShuttleBuilder(...) {
    return InheritedTheme.captureAll(
        heroContext,
        Material(type: MaterialType.transparency, child: hero.child),
    );
}
```

## 11.5 KazumiDialog — 全局对话框系统

> 源码：`lib/bean/dialog/dialog_helper.dart`

```dart
class KazumiDialog {
    // 通用对话框
    static void show({required Widget Function(BuildContext) builder});

    // Toast（SnackBar）
    static void showToast({required String message});

    // 加载中对话框
    static void showLoading({String? msg, bool barrierDismissible = false});

    // 底部弹窗
    static void showBottomSheet({required Widget Function(BuildContext) builder});

    // 关闭当前对话框
    static void dismiss();

    // 自动关闭的成功对话框（带倒计时）
    static void showTimedSuccessDialog({required String title, required String message, VoidCallback? onComplete});
}
```

**关键设计：通过 `KazumiDialogObserver`（NavigatorObserver）全局追踪路由上下文，不需要传递 BuildContext 就能在任何地方弹对话框。**

## 11.6 BBCodeWidget — BBCode 富文本渲染

> 源码：`lib/bbcode/bbcode_widget.dart`

Bangumi 社区使用 BBCode 标记语言，Kazumi 用 ANTLR4 实现了解析器。

### 支持的 BBCode 标签

| 标签 | 说明 | 渲染效果 |
|------|------|----------|
| `[b]文字[/b]` | 粗体 | `FontWeight.bold` |
| `[i]文字[/i]` | 斜体 | `FontStyle.italic` |
| `[u]文字[/u]` | 下划线 | `TextDecoration.underline` |
| `[s]文字[/s]` | 删除线 | `TextDecoration.lineThrough` |
| `[color=red]文字[/color]` | 颜色 | 支持 #RRGGBB、#AARRGGBB、颜色名 |
| `[size=20]文字[/size]` | 字号 | `fontSize` |
| `[url=链接]文字[/url]` | 超链接 | 蓝色 + 下划线 + 点击打开 |
| `[user=用户名]` | 用户链接 | 蓝色 + 下划线 |
| `[img]图片URL[/img]` | 图片 | `CachedNetworkImage` + 点击全屏 |
| `[mask]剧透文字[/mask]` | 剧透 | 默认透明 + 灰色背景，点击切换显示 |
| `[quote]引用文字[/quote]` | 引用 | 灰色文字 |
| `[bgm-42]` | Bangumi 表情 | 从 bangumi.tv 加载图片 |
| `[musume-42]` | Bangumi 娘表情 | 从 lain.bgm.tv 加载 |
| `[sticker-42]` | 贴纸 | 从 bangumi.tv 加载 |

### 渲染流程

```
"[b]好看[/b] [color=red]推荐[/color]"
    ↓
ANTLR4 Lexer → Token 流
    ↓
ANTLR4 Parser → AST 语法树
    ↓
BBCodeBaseListener → 遍历 AST → 构建元素列表
  [BBCodeText(bold=true, "好看"), BBCodeText(color="red", "推荐")]
    ↓
BBCodeWidget → Wrap + RichText
  [粗体"好看"] [红色"推荐"]
```

## 11.8 ScaffoldMenu — 自适应导航

> 源码：`lib/pages/menu/menu.dart`

### 竖屏布局

```
┌──────────────────────┐
│                      │
│    PageView          │
│    (当前Tab页面)      │
│                      │
│──────────────────────│
│ 🏠  📅  📚  👤      │  ← NavigationBar（底部）
└──────────────────────┘
```

### 横屏布局

```
┌────┬─────────────────┐
│ 🏠 │                 │
│ 📅 │   PageView      │
│ 📓 │   (当前Tab页面)  │
│ 👤 │                 │
│    │                 │
│ 🔍 │                 │  ← NavigationRail（侧边）
└────┴─────────────────┘
```

### 关键代码

```dart
class ScaffoldMenu extends StatelessWidget {
    Widget build(BuildContext context) {
        return OrientationBuilder(
            builder: (context, orientation) {
                if (orientation == Orientation.portrait) {
                    // 竖屏：底部 NavigationBar
                    return _buildBottomNavigation();
                } else {
                    // 横屏：侧边 NavigationRail
                    return _buildSideNavigation();
                }
            },
        );
    }
}
```

## 11.11 与 Java/Vue 项目的对比

| 概念 | Kazumi (Flutter) | Vue | React |
|------|------------------|-----|-------|
| 组件化 | Widget 类 | SFC (.vue) | 函数组件 |
| 状态管理 | MobX @observable | Vuex/Pinia | useState/Redux |
| 路由 | flutter_modular | Vue Router | React Router |
| 图片缓存 | CachedNetworkImage | vue-lazyload | react-lazyload |
| 富文本 | ANTLR4 + RichText | v-html | dangerouslySetInnerHTML |
| 自适应布局 | OrientationBuilder | CSS Media Query | CSS Media Query |
| 对话框 | KazumiDialog (全局) | Element Plus | Ant Design |
