# 01 - 项目概览

## 技术栈

| 类别 | 技术选型 |
|------|----------|
| 框架 | Flutter 3.44.1, Dart 3.12.1 |
| 路由/DI | flutter_modular |
| 状态管理 | MobX + Provider (仅主题) |
| 网络 | Dio (4 实例策略) |
| 本地存储 | Hive CE |
| 视频播放 | media_kit (自定义 fork) |
| 弹幕渲染 | canvas_danmaku |
| 代码生成 | mobx_codegen, hive_ce_generator |

## 架构分层

```
┌─────────────────────────────────────────────────┐
│                   UI Layer                       │
│  Page (Widget) <- Observer <- Controller (MobX) │
├─────────────────────────────────────────────────┤
│               Business Layer                    │
│  Service (Player/Download/Sync/Plugin)          │
├─────────────────────────────────────────────────┤
│                Data Layer                       │
│  Repository (Interface) -> Hive (Storage)       │
├─────────────────────────────────────────────────┤
│              Network Layer                      │
│  API (Static) -> Client (Singleton) -> Dio      │
└─────────────────────────────────────────────────┘
```

## 核心设计模式

| 模式 | 应用场景 |
|------|----------|
| Singleton | 全局服务单例 (BangumiClient, WebDav) |
| Factory | 平台 WebView 创建 |
| Facade | 播放器协调 (PlayerController) |
| Observer | MobX 响应式状态 |
| Repository | 数据访问抽象 |
| Interceptor | Dio 请求拦截 |
| Composition | 播放器 6 个子控制器协作 |
