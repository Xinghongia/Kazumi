## 启动链路总览

从 `main.dart` 出发，7 个根文件的完整调用链：

```
main.dart                  ← 入口，初始化基础设施
    │
    │  runApp(ModularApp(module: AppModule(), child: AppWidget()))
    │
    ├─ AppModule           ← 路由根，转发给 IndexModule
    │
    ├─ AppWidget           ← UI 根，MaterialApp.router 接入路由
    │
    ▼
IndexModule                ← 路由中枢，定义所有路由 + 注册所有依赖
    │
    ├─ router.dart         ← Tab 路由注册表 (menu 变量)
    │
    ├─ routes:
    │    ├─ "/" → InitPage ← 启动页，执行应用初始化
    │    │                  初始化完成后跳转到 "/tab/popular/"
    │    │
    │    └─ "/tab" → IndexPage ← 导航壳，底部栏/侧边栏
    │         └─ children: menu.routes
    │              ├─ "/popular"  → PopularPage
    │              ├─ "/timeline" → TimelinePage
    │              ├─ "/collect"  → CollectPage
    │              └─ "/my"       → MyPage
    │
    └─ 其他路由:
         ├─ "/video"    → VideoPage
         ├─ "/info"     → InfoPage
         ├─ "/settings" → SettingsPage
         └─ "/search"   → SearchPage
```

## 每个文件的职责

| 文件 | 职责 | 一句话 |
|------|------|--------|
| `main.dart` | 入口 | 初始化引擎/数据库/窗口，启动应用 |
| `AppModule` | 路由根 | 只做转发，全部交给 IndexModule |
| `AppWidget` | UI 根 | MaterialApp.router，主题/托盘/窗口 |
| `IndexModule` | 路由中枢 | 定义所有路由 + 注册所有依赖 |
| `router.dart` | Tab 注册表 | 4 个 Tab 的路径和模块 |
| `InitPage` | 启动页 | 加载插件/同步/着色器，然后跳转首页 |
| `IndexPage` | 导航壳 | 底部栏/侧边栏 + PageView |

## 文件之间的关系

### 路由链

```
AppModule
    │  r.module("/", module: IndexModule())
    ▼
IndexModule
    │  routes(r) 定义所有路由
    ▼
router.dart
    │  menu 变量定义 4 个 Tab
    ▼
PopularModule / TimelineModule / CollectModule / MyModule
    │  每个模块定义自己的子路由
    ▼
PopularPage / TimelinePage / CollectPage / MyPage
```

### UI 链

```
AppWidget
    │  MaterialApp.router(routerConfig: Modular.routerConfig)
    │  Modular 根据当前路径，渲染对应的页面
    ▼
InitPage（第一个显示的页面）
    │  初始化完成后跳转
    ▼
IndexPage（导航壳）
    │  PageView + RouterOutlet
    ▼
PopularPage / TimelinePage / CollectPage / MyPage
```


## 每个文件的详细说明

### 1. main.dart — 入口

```
main()
  ├─ WidgetsFlutterBinding.ensureInitialized()  ← Flutter 引擎
  ├─ MediaKit.ensureInitialized()               ← 视频播放器
  ├─ Hive.initFlutter() + GStorage.init()       ← 本地数据库
  ├─ windowManager.ensureInitialized()          ← [Desktop] 窗口
  ├─ ProxyManager.applyProxy()                  ← 代理设置
  └─ runApp(
       ChangeNotifierProvider<ThemeProvider>     ← 主题
         └─ ModularApp<AppModule>               ← DI + 路由
              └─ AppWidget                      ← UI 根
     )
```

源码位置：`lib/main.dart`

### 2. AppModule — 路由根

```dart
class AppModule extends Module {
  void binds(i) {}  // 空的，不注册依赖

  void routes(r) {
    r.module("/", module: IndexModule());  // 全部转发
  }
}
```

源码位置：`lib/app_module.dart`

### 3. AppWidget — UI 根

```dart
class AppWidget extends StatefulWidget {

  Widget build(context) {
```

源码位置：`lib/app_widget.dart`

### 4. IndexModule — 路由中枢

```dart
class IndexModule extends Module {
  // 导入 Tab 模块的依赖
  List<Module> get imports => menu.moduleList;

  // 注册所有依赖（15 个单例）
  void binds(i) {
    // Repository 层 (5 个)
    // Service 层 (2 个)
    // Controller 层 (8 个)
  }

  // 定义所有路由
  void routes(r) {
    r.child("/", child: InitPage());          // 启动页
    r.child("/tab", child: IndexPage(),        // 导航壳
      children: menu.routes);                  // 4 个 Tab
    r.module("/video", module: VideoModule());  // 播放器
    r.module("/info", module: InfoModule());    // 详情页
    r.module("/settings", ...);                 // 设置
    r.module("/search", ...);                   // 搜索
  }
}
```

源码位置：`lib/pages/index_module.dart`

### 6. InitPage — 启动页

```dart
class InitPage extends StatefulWidget {
  // 职责：执行应用初始化，然后跳转首页
  // build() 只返回空 Scaffold，不做 UI 渲染

  Future<void> _initializeApp() async {
    _migrateStorage();           // 1. 数据迁移
    _loadShaders();              // 2. 着色器
    _loadDanmakuShield();        // 3. 弹幕屏蔽
    _webDavInit();               // 4. WebDAV 同步
    _bangumiInit();              // 5. Bangumi 同步
    downloadController.init();   // 6. 下载管理器
    _checkRunningOnX11();        // 7. X11 检测
    _pluginInit();               // 8. 插件初始化 + 免责声明
    _showShortcutDialog();       // 9. 桌面快捷方式
    _startDefaultPage();         // 10. ★ 跳转到首页
    _update();                   // 11. 检查更新
  }
}
```

源码位置：`lib/pages/init_page.dart`

## 核心设计思想

### 1. 关注点分离

| 文件 | 关注点 |
|------|--------|
| `main.dart` | 基础设施初始化 |
| `AppModule` | 路由结构 |
| `AppWidget` | UI 框架 |
| `IndexModule` | 依赖注册 + 路由定义 |
| `router.dart` | Tab 配置 |
| `InitPage` | 应用初始化 |
| `IndexPage` | 导航布局 |

### 2. 单一职责

- `AppModule` 只做转发，不注册依赖
- `AppWidget` 只管 UI 框架，不管业务
- `IndexModule` 集中管理所有依赖和路由
- `router.dart` 只定义 Tab，不管其他路由

### 3. 渐进式初始化

```
main.dart     → 基础设施（引擎、数据库、窗口）
InitPage      → 应用功能（插件、同步、着色器）
IndexPage     → UI 渲染（导航栏、页面）
PopularPage   → 业务内容（番剧列表）
```

每一层只做自己的事，完成后交给下一层。
